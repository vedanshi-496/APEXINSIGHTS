# **App Name**: Apex Insights

## Core Features:

- Project Display: Display projects on the landing page, fetched from the backend, including image, name, and description.
- Client Display: Display client information on the landing page, fetched from the backend, including image, description, name, and designation.
- Contact Form Submission: Allow users to submit contact form details (full name, email, mobile, city) and store the submissions in the database.
- Newsletter Subscription: Capture user email addresses for newsletter subscriptions and store them in the database.
- Project Management (Admin): Enable admin to add, edit, and delete projects with image, name, and description.
- Client Management (Admin): Enable admin to add, edit, and delete client information with image, name, description, and designation.
- View Contact Form Details (Admin): Display all contact form submissions in the admin panel.
- View Subscribed Emails (Admin): Display all subscribed email addresses in the admin panel.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey trust and professionalism.
- Background color: Light gray (#F0F0F0) to provide a clean and modern look.
- Accent color: Electric Purple (#BF00FF) for highlighting key elements and calls to action.
- Body font: 'PT Sans', sans-serif.
- Headline font: 'Space Grotesk', sans-serif.
- Use professional and clear icons to represent different sections and actions.
- Maintain a clean and organized layout with clear sections and visual hierarchy.
- Implement subtle transitions and animations for a smoother user experience.
